# ActivityWheel

A colorful static website for choosing family activities with a spinning wheel.

## Features

- Spin a wheel to randomly choose an activity
- Edit the activity list directly in the browser
- Save activities in local storage on the device
- Works as a simple static site that can be hosted with GitHub Pages

## Run locally

Open `index.html` in a browser.

## Publish

This repo is ready to push to GitHub and host as a static website, for example with GitHub Pages.
